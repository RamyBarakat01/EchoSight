## EchoSight:

EchoSight is a 2D mobile stealth-action game built with Unity, where the player navigates a dark environment using sound as their primary source of vision.
In EchoSight, the player controls a blind swordsman who cannot see the world normally. Instead, visibility is limited to a small area around the player. The environment is revealed temporarily through sound-based pulses triggered by movement, attacks, and interactions.
Enemies can also generate sound, allowing the player to detect their position indirectly. This creates a unique gameplay experience focused on awareness, timing, and spatial reasoning.

## Core Mechanics:

Limited Vision: Player can only see a small radius around them.
Sound Pulses: Sounds generate temporary visual pulses that reveal the environment.
Combat System: Melee attacks with area-based detection.
Dash Mechanic: Allows quick movement and evasion.
Enemy AI:
  Patrol and chase behavior
  Sound-based detection through footsteps
  Heavy enemies with higher damage and slower movement
Health System: Player has limited health displayed using a heart-based UI.

##  Platform:

Mobile (Android-focused design)
Also playable in Unity Editor using keyboard controls

##  Technologies Used:

Game Engine: Unity 6 (6000.3.9f1 LTS)
Language: C#
IDE: Unity Editor + Visual Studio / VS Code
UI System: Unity UI + TextMeshPro
Physics: Unity 2D Physics Engine
Audio: Unity Audio Source system
